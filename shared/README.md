# shared — единственная копия общего кода фронтендов

Волна 2 правила №3 CLAUDE.md: файлы, обязанные совпадать в webapp и
schema-miniapp, живут здесь; в обоих деревьях лежат однострочные реэкспорты
(пути идентичны — драйв-гард парности продолжает их сверять).

Без npm-пакета осознанно: Docker-сборка webapp идёт из полного контекста
(`COPY . .`), vite резолвит относительные импорты — ничего в деплое менять
не нужно. Мини-апп собирается локально — аналогично.

НЕ переносить сюда файлы с фронтенд-специфичными импортами (`../api` и т.п.):
addressForm.tsx и CrisisCard.tsx остались парными копиями именно поэтому.
Новому общему коду место здесь, а не в копипасте.
