# shared — единственная копия общего кода фронтендов

Волна 2 правила №3 CLAUDE.md: файлы, обязанные совпадать в webapp и
schema-miniapp, живут здесь; в обоих деревьях лежат однострочные реэкспорты
(пути идентичны — драйв-гард парности продолжает их сверять).

Без npm-пакета осознанно: Docker-сборка webapp идёт из полного контекста
(`COPY . .`), vite резолвит относительные импорты — ничего в деплое менять
не нужно. Мини-апп собирается локально — аналогично.

НЕ переносить сюда файлы с фронтенд-специфичными импортами (`../api` и т.п.):
addressForm.tsx и CrisisCard.tsx остались парными копиями именно поэтому.
Новому общему коду место здесь, а не в копипасте.

## Волна 2 (выравнивание почти-пар)

- `utils/format.ts`, `utils/storageKeys.ts` — разошлись только форматированием
  кода (плюс storageKeys.ts у webapp содержал доп. константы, не вынесенные
  у мини-аппа в отдельный util) — выровнены и перенесены сюда целиком.
- `utils/celebrationText.ts` — вынесена только текстовая логика Celebration
  (`pluralDays`, `getMilestoneText`, `MILESTONE_TEXT`); сама вёрстка/canvas-
  анимация остаётся в `components/Celebration.tsx` каждого фронтенда отдельно
  — там намеренное визуальное расхождение (fontWeight стрик-числа: 600 у
  webapp, 800 у мини-аппа), поэтому весь компонент не переносится.
- `components/TherapyNote.tsx` — сравнение показало только дрейф
  форматирования кода и типа тире «–»/«—» в тексте, логика (и даже сам текст,
  не считая тире) идентична. Не перенесён в shared: импортирует `useTr` из
  `../utils/addressForm`, который сам фронтенд-специфичный (см. выше) и не
  может резолвиться из shared. Побайтовое выравнивание пары тоже не
  применено — проверено: `jscpd`-храповик считает выровненную пару целиком
  новым клоном ≥70 токенов (99 строк/422 токена, т.к. до выравнивания
  разные тире разбивали совпадение на кусочки ниже порога), и итоговый счётчик
  дублированных строк растёт вопреки остальным правкам волны 2, которые его
  снижают. Остаётся задокументированным незначительным дрейфом (см.
  `scripts/check-paired-files.mjs` комментарий), а не выровненной парой.
